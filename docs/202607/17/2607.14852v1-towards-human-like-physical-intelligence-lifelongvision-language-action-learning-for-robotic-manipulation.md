## 摘要
类似于人类顺序学习新任务的自然能力，配备视觉-语言-动作（VLA）模型的机器人在开放世界环境中部署时应具备终身学习能力，以学习新任务。然而，最近提出的大多数终身学习模型旨在有效学习当前任务（可塑性）或保持先前任务的高精度（稳定性），而机器人操作模型中的可塑性-稳定性权衡仍未得到很好解决。为应对这一基本挑战，我们提出了一种用于机器人操作的缓存高效终身视觉-语言-动作学习框架（即LifelongVLA），该框架通过双时间尺度适应机制缓解可塑性-稳定性权衡，同时通过缓存高效的重放策略实现低成本的机器人部署。具体而言，我们提出了一种双时间尺度LoRA门控模块，将VLA适应分解为两个轻量级路径：用于可塑性的短期适配器和用于稳定巩固的长期适配器。这些路径通过任务感知门进行集成，从而实现对可塑性-稳定性权衡的显式控制。在技能重放阶段，提出了一种缓存高效的随机重放策略，无需存储完整轨迹即可保留更平衡的保留信号。最后，实验表明LifelongVLA优于现有基线，展示了高效的技能扩展、对已学习操作行为的稳健保留，以及在xArm机器人上实际部署时对重新训练的依赖减少。

## Abstract
Similar to the natural capabilities of humans to sequentially learn new tasks, robots with Vision-Language-Action (VLA) models should possess lifelong learning ability to learn a new task when deployed in open-world environments. However, most recently proposed lifelong learning models aim to effectively learn the current task (plasticity) or maintain high accuracy on previous tasks (stability), while the plasticity-stability trade-off remains largely unsolved in robotic manipulation models. To address this fundamental challenge, we propose a cache-efficient lifelong Vision-Language-Action learning framework for robotic manipulation (i.e., LifelongVLA), which alleviates the plasticity-stability trade-off with a dual-timescale adaptation mechanism while achieving low-cost robotic deployment with a cache-efficient replay strategy. More concretely, we propose a dual-timescale LoRA gating module to decompose VLA adaptation into two lightweight pathways: a short-term adapter for plasticity and a long-term adapter for stable consolidation. These pathways are integrated via a task-aware gate, enabling explicit control of the plasticity-stability trade-off. In the skill replay phase, a cache-efficient stochastic replay strategy is proposed to preserve more balanced retention signals without full-trajectory storage. Finally, experiments show that LifelongVLA outperforms existing baselines, demonstrating efficient skill expansion, robust retention of learned manipulation behaviors, and reduced reliance on retraining for real-world deployment on an xArm robot.

## 1. 论文的核心问题与整体含义

论文聚焦于机器人操作中的**终身视觉-语言-动作（VLA）学习**问题。现有VLA模型（如RT系列、Octo、π0等）通常在固定数据集上离线训练一次后部署，缺乏持续学习新技能的能力。当机器人在开放世界中连续部署时，新任务不断涌现，直接顺序微调会导致**灾难性遗忘**（对先前已学技能的精度急剧下降），而简单保留所有历史数据并重训练成本过高。该问题的本质是**可塑性-稳定性权衡**：如何让模型快速适应新任务（高可塑性）的同时保留旧任务知识（高稳定性）。目前终身学习方法（如重放、参数隔离、蒸馏）仍未能充分解决这一权衡，且重放策略对VLA模型（图像密集、骨干网络大）存储和计算开销极大。

## 2. 论文提出的方法论

### 核心思想
提出 **LifelongVLA** 框架，在冻结骨干网络上通过两类轻量级参数高效微调（LoRA）模块分别建模短期可塑性与长期稳定性，并结合缓存高效的随机重放策略降低存储和计算开销。

### 关键技术细节

#### （1）双时间尺度LoRA门控模块（Dual-Timescale LoRA Gating）
- 为每个适配层定义两条LoRA路径：
  - **短期适配器** `∆W_sh`：使用当前任务损失 `L_new` 更新，学习速率 `η_sh` 较大，负责快速吸收新技能（高可塑性）。
  - **长期适配器** `∆W_lg`：仅使用重放损失 `L_rep` 和蒸馏损失 `L_dist` 更新，学习速率 `η_lg` 较小（`η_lg < η_sh`），负责巩固已学技能（高稳定性）。
- **任务感知门控**：门控上下文 `c` 从冻结骨干的池化特征（停止梯度）计算，通过轻量网络预测门控值 `α(c) ∈ [0,1]`，用于在权重空间组合两条路径：`∆_cW = (1-α)∆W_sh + α∆W_lg`。当 `α→0` 强调短期可塑性，`α→1` 强调长期稳定性。门控对当前样本和重放样本共享。
- 最终权重：`cW = W_0 + γ·∆_cW`，仅需单次前向推理。

#### （2）缓存高效随机重放模块（Cache-Efficient Stochastic Replay）
- **缓存存储**：每个新任务随机选取少量样本（如50个），仅存储停止梯度的**前缀token**（来自视觉-语言编码器的输出，不存原始图像）以及紧凑的状态-动作监督。不存储完整轨迹、后缀token或扩散噪声。
- **随机重放**：从缓存中采样时，为每个样本重新采样扩散时间步 `τ'` 和噪声 `ε'`，并使用当前模型重新计算后缀token `h_suf`。因此少量缓存可生成多样化的重放实例。
- **损失设计**：当前样本与重放样本混合在同一batch中，分别计算 `L_new` 和 `L_rep`（使用回归损失）。此外，利用任务 `t-1` 时的模型快照作为教师，对重放样本施加蒸馏损失 `L_dist`（L2距离），稳定旧技能预测。总损失：`L = L_new + λ·L_rep + β·L_dist`。

## 3. 实验设计

### 数据集/场景与基准（Benchmark）
- **仿真实验**：使用 **LIBERO** 基准中的10个操作任务（如放置物体、拾取碗等），每个任务作为一个增量步骤顺序学习。任务输入为语言指令、视觉观测和机器人状态，输出为连续7维动作。
- **真实机器人实验**：使用 **xArm** 机器人，5个真实操作任务（如“将茄子放在盘子上”、“将黑碗放在盘子上”等），每个任务收集50个训练演示，50个评估演示。

### 对比方法
- **SFT**：顺序微调所有参数。
- **LwF-LoRA**：知识蒸馏 + LoRA。
- **ER**：经典经验重放（存储完整轨迹）。
- **Info-VLA**：信息论约束的VLA终身学习。

### 评估指标
- **成功率（SR）**：训练完所有任务后，每个任务的测试成功率，并取平均。
- **遗忘率（FOR）**：每个旧任务在后续训练中精度下降的幅度，取平均。
- **过程级精度（ACC_t）**：每个增量步骤后已见任务的平均成功率。

## 摘要
我们研究动作条件世界建模作为机器人学习中一种可扩展的学习可转移动力学先验的方法。通过预训练一个模型来预测动作如何驱动视觉场景演化，所得到的世界模型捕捉了超越外观级视频生成的可复用交互动力学。具体来说，我们在具有真实动作标注的大规模机器人操作数据上预训练了一个多视角交互基础扩散世界模型A2World。我们从两个互补的角度验证了学习到的动力学先验。首先，我们将A2World适配为任务或场景专用的真实世界模拟器A2World-sim，其长期展开通过用世界模型展开替代真实机器人展开，支持基于模拟器的策略评估和可扩展的假设分析。其次，从相同的预训练权重出发，我们将A2World适配为视频-动作联合预测模型A2World-policy，该模型在视觉和指令条件下预测动作。在仿真基准和真实机器人环境中的实验表明，动作条件世界模型预训练产生了可转移的动力学先验，有利于以模拟器为中心和以策略为中心的机器人学习。

## Abstract
We study action-conditioned world modeling as a scalable way to learn transferable dynamics priors for robot learning. By pretraining a model to predict how actions drive visual scene evolution, the resulting world model captures reusable interaction dynamics beyond appearance-level video generation. Concretely, we pretrain a multi-view interactive base diffusion world model, A2World, on large-scale robot manipulation data with real action annotations. We validate the learned dynamics priors from two complementary perspectives. First, we adapt A2World into a task- or scene-specialized real-world simulator, A2World-sim, whose long-horizon rollouts support simulator-based policy evaluation and scalable what-if analysis by replacing real-robot rollouts with world model rollouts. Second, starting from the same pretrained weights, we adapt A2World into a video-action joint prediction model, A2World-policy, that predicts actions under visual and instruction conditioning. Experiments across simulation benchmarks and real-robot settings demonstrate that action-conditioned world model pretraining yields transferable dynamics priors that benefit both simulator-centric and policy-centric robot learning.

#### 2. 方法论：核心思想与关键技术细节

-   **核心思想**：通过在大规模、多机器人数据集上预训练一个动作条件的多视图扩散世界模型，使其学习到通用的“动作→视觉状态变化”的映射，并将此作为可迁移的先验知识。
-   **技术框架与模型设计**：提出了**A2World**系列模型，包含三个变体：
    -   **A2World (基础世界模型)**:
        -   **输入**: 一个初始帧 $o_t$ 和一个未来动作块 $a_{t+1:t+k}$。
        -   **输出**: 预测的未来观测 $o_{t+1:t+k}$。
        -   **架构**: 基于 Diffusion Transformer (DiT) 的潜视频扩散模型。关键模块：
            -   **动作注入**: 将未来动作编码后，加到扩散时间步嵌入中，通过自适应层归一化 (AdaLN) 影响每个DiT块，实现动作控制。
            -   **多视图生成**: 将多个摄像机视图沿时间维度拼接，通过**可学习视图嵌入**区分不同视角，并在DiT块中插入**跨视图注意力模块**以保持多视图时空一致性。
            -   **预训练**: 使用EDM去噪分数匹配目标进行训练。
    -   **A2World-sim (长期模拟器)**:
        -   **功能**: 从A2World初始权重微调而来，用于长期的、自回归的视频预测。它支持基于历史的模拟。
        -   **关键技术**:
            1.  **基于位姿的历史采样**: 通过计算历史动作序列的加权弧长，均匀采样少量关键帧作为历史信息（称为“历史令牌”），解决了长期模拟中历史信息冗余和效率问题。
            2.  **历史注入**: 将采样到的历史令牌通过交叉注意力和自注意力内存路径注入DiT块，为模型提供丰富的上下文信息。
            3.  **自回归生成与Self-forcing训练**: 采用分块自回归方式生成长序列，并使用Self-forcing训练策略（用模型自身生成的帧作为下一个时间步的条件），以缓解训练与推理之间的分布偏移，提高长期稳定性。
    -   **A2World-policy (视频-动作联合预测策略)**:
        -   **功能**: 从A2World初始权重微调而来，成为一个**指令条件**的机器人策略。它能够联合预测未来视频和动作。
        -   **关键技术**:
            1.  **MoE-like 设计**: 采用类似专家混合思想，视频令牌和动作令牌**共享一个自注意力模块**进行交互，但各自拥有独立的 AdaLN 和 MLP（多层感知机）分支，实现专有建模。
            2.  **联合扩散公式**: 对视频潜变量和动作潜变量独立添加噪声，使用**共享时间步**（不同缩放因子）进行联合去噪训练，有助于对齐视频和动作模态。
            3.  **模态导向的无分类器引导**: 推理时，对视频和动作使用不同的引导尺度（$s_v, s_a$），可以灵活权衡视觉质量和动作准确性。

#### 6. 主要结论与发现

1.  **动作条件预训练优于文本条件预训练**：在动作条件世界模型（A-pre）上获得的先验，在进行下游策略微调时，无论是在分布内还是分布外，都显著优于直接从通用视频模型（C-init）或从文本条件世界模型（T-pre）初始化的方法。这表明**动作**是一种比**文本**更强、更明确的动力学监督信号。
2.  **统一先验的双重效用**：从**同一个**动作条件预训练权重出发，可以高效地适应为**高质量模拟器**（A2World-sim）和**高性能策略**（A2World-policy）。这证明了该先验的通用性和可移植性，实现了“一次预训练，双重受益”。
3.  **视频与动作的协同作用**：在A2World-policy中，视频预测和动作生成的性能在整个训练过程中呈正相关。更好的视频一致性伴随着更好的动作质量，说明共享的视觉动力学表示能够相互促进，共同提升。
4.  **长期模拟的稳定性**：通过历史采样和Self-forcing训练，A2World-sim能够生成稳定且可信的长期动作条件视频，其成功率与真实世界的成功率高度相关（Pearson r = 0.965），证明了其作为可靠策略评估器的潜力。
5.  **跨实体和多视图泛化**：A2World能够处理来自不同视觉设定（如不同视角数量）的数据，并通过视图嵌入和跨视图注意力保持一致的多视图输出，展现出良好的跨实体和场景泛化能力。

#### 7. 优点

1.  **范式创新**：明确提出了“动作条件世界模型预训练”作为一种可学习的、可迁移的动力学先验，为连接模拟与原型的机器人学习提供了一种统一的、可扩展的范式。
2.  **规模宏大**：预训练数据规模大（210万+轨迹）、覆盖广泛（20+机器人实体），是首个在如此大规模动作标注数据集上预训练的世界模型。
3.  **设计精巧**：针对模拟器和策略的不同需求，设计了两种不同的适配变体，并引入了多项有效技术（如基于位姿的历史采样、Self-forcing训练、MoE-like联合模型）。

#### 8. 不足与局限

1.  **数据依赖与偏差**：预训练和微调高度依赖高质量、带动作标注的机器人数据。尽管数据量大，但仍可能**存在数据偏差**，例如对某些特定物体或动作pattern的过度学习，这可能会影响其在完全未见过的场景或物体上的泛化。
2.  **动作空间通用化**：论文将不同机器人的动作统一为一个共享7维向量格式，但这可能**牺牲了某些复杂机械臂（如灵巧手）的精细动作精度**。对于非标准动作空间的适应性是一个局限。
3.  **计算资源需求高**：整套系统的训练和微调需要**大规模GPU集群（64/32张H200）**，对于资源有限的研究团队或应用场景，复现门槛较高。
4.  **真实机器人长期模拟的验证范围有限**：在真实机器人上验证模拟器一致性的实验仅覆盖了5个任务和2个策略（共8个数据点），虽然结果积极，但**样本量较小**，结论的普适性有待在更多任务和策略上进一步验证。
5.  **安全性与鲁棒性**：作为世界模型和策略，其生成的内容（视频、动作）在真实世界部署中可能存在危险。论文未深入探讨模型在此类安全关键场景下的鲁棒性和失败率，这是一个重要的实际应用限制。
6.  **长期滚动中的潜在误差累积**：虽然Self-forcing训练缓解了部分问题，但在极其长期（如数百乃至上千帧）的模拟中，误差累积仍然是潜在风险，论文未进行此类极端长程的实验。

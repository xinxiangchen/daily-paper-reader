## 摘要
离散动作标记化为自回归VLA策略提供了紧凑的接口，但从离散编码中准确恢复连续机器人动作仍然具有挑战性。现有标记器通常将每个离散编码映射到固定的连续动作原型，忽略了机器人当前的自身感知状态。这一局限在操作任务中尤为突出，因为相同的动作标记在不同关节构型、物体姿态和接触条件下可能需要不同的连续控制。为此，我们提出SA-VLA，一种面向状态的动作标记器，它根据机器人状态来条件化动作解码。我们研究了两种基于VQ的动作标记化的状态注入机制：状态与动作特征之间的交叉注意力，以及一个轻量级状态适配器，该适配器预测动作级的调制因子，用于状态条件化的动作调制和重建。适配器公式通过允许每个离散标记表示一族依赖于状态的连续动作，扩展了有限码书的有效支持范围，同时保留了离散动作建模的效率和兼容性。SA-VLA集成到基于LLM的VLA策略中，支持自回归和平行动作标记解码，且对模型接口的改动极小。在12个RoboTwin操作任务上，SA-VLA相较于最强标记器基线，将平均成功率从0.29提升至0.56。在三个真实世界任务的零样本仿真到真实实验中，它进一步将平均成功率从0.15提升至0.33。这些结果表明，状态条件化的动作解码是一种简单有效的机制，能够缩小离散VLA策略中的压缩差距。

## Abstract
Discrete action tokenization provides a compact interface for autoregressive VLA policies, but accurately recovering continuous robot actions from discrete codes remains challenging. Existing tokenizers typically map each discrete code to a fixed continuous action prototype, ignoring the robot's current proprioceptive state. This limitation is particularly pronounced in manipulation, where the same action token may require different continuous controls under different joint configurations, object poses, and contact conditions. We therefore propose SA-VLA, a state-aware action tokenizer that conditions action decoding on robot state. We study two state-injection mechanisms for VQ-based action tokenization: cross-attention between state and action features, and a lightweight state adapter that predicts action-wise modulation factors for state-conditioned action modulation and reconstruction. The adapter formulation expands the effective support of a finite codebook by allowing each discrete token to represent a family of state-dependent continuous actions, while preserving the efficiency and compatibility of discrete action modeling. Integrated into an LLM-based VLA policy, SA-VLA supports both autoregressive and parallel action-token decoding with minimal changes to the model interface. On 12 RoboTwin manipulation tasks, SA-VLA improves the average success rate from 0.29 to 0.56 over the strongest tokenizer baseline. In zero-shot sim-to-real experiments on three real-world tasks, it further improves average success from 0.15 to 0.33 over the strongest tokenizer baseline. These results demonstrate that state-conditioned action decoding is a simple and effective mechanism for reducing the compression gap in discrete VLA policies.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：离散动作标记化（discrete action tokenization）为自回归 VLA（视觉-语言-动作）策略提供了紧凑接口，但现有方法通常将每个离散编码映射到一个固定的连续动作原型，**忽略了机器人当前的自身感知状态**（如关节角度、物体位姿、接触条件等）。在操控任务中，相同的动作令牌（action token）在不同状态下需要不同的连续控制，导致动作恢复准确性严重受限。
- **研究动机**：引入机器人状态信息来条件化动作解码，使每个离散令牌能够表示一族依赖于状态的连续动作，从而缩小离散化带来的“压缩差距”（compression gap）。
- **整体含义**：提出 SA-VLA（State-Aware Vision-Language-Action）模型，通过两种状态注入机制改进 VQ-VAE 基础的动作标记器，显著提升离散 VLA 策略的操控成功率。

## 2. 方法论

### 核心思想
在 VQ-VAE 动作标记器的编码和解码过程中，注入机器人的当前状态信息，使解码出的连续动作能够根据状态自适应调整，而不是输出固定原型。

### 关键技术细节（两种注入机制）

#### Method A：交叉注意力（Cross-Attention）
- 在编码器和解码器的 Transformer 层中，将状态特征作为 Key 和 Value，与动作特征进行交叉注意力交互。
- 编码器输出中间特征 `Z ∈ R^{R×d}`，经 VQ 量化得到离散嵌入，再通过对称解码器（同样使用状态作为 Key/Value）重建出连续动作 `ˆa`。
- 损失函数为标准 VQ-VAE 损失：重建损失 + 码书损失 + 承诺损失。

#### Method B：轻量级状态适配器（State Adapter）
- 状态信息首先通过一个由 MLP 和 Sigmoid 激活函数组成的轻量级适配器，输出一个**缩放因子向量** `w`，其维度与动作每个维度的长度一致。
- 动作为缩放变换：`a_trans = a ÷ w`（逐元素除法）。
- 将变换后的动作 `a_trans` 送入标准的 VQ-VAE（编码器/解码器中**不使用状态信息**），得到重建的 `ˆa_trans`。
- 最终重建动作：`ˆa = ˆa_trans × w`（保持对称性）。
- **优势**：每个离散令牌不再对应唯一动作，而是通过状态适配器生成连续的缩放空间，扩展了有限码书（codebook）的有效表示能力，本质上是将离散 VQ 转化为回归问题。

### 集成到 VLA 中
- 以 Qwen2.5 为基础 VLM，将图像（SigLIP 编码）、语言、状态（离散化为256个bin）、动作（经 SA-VLA 标记器）以固定格式拼接成序列，用特殊标记分隔。
- 支持两种解码方式：
  - **自回归解码**：逐个预测动作令牌（负对数似然损失）。
  - **平行解码**：使用双向注意力一次性生成全部动作令牌，推理效率更高。

1. **状态条件化动作解码显著优于传统固定原型解码**：Method B（适配器）在12个RoboTwin任务上平均成功率高达**56%**，比最强基线VQ-BET（29%）提升**27个百分点**，比binning（24%）提升32个百分点。

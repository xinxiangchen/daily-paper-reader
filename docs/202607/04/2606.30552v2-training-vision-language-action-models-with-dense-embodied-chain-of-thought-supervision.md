## 摘要
交叉具身迁移在视觉-语言-动作（VLA）模型中仍然具有挑战性，因为低级状态和动作空间在不同机器人平台上存在根本差异。我们观察到，操纵背后的高级认知过程，包括场景感知、物体识别、任务规划和子任务分解，在各种具身形态中很大程度上是共享的。基于这一观察，我们提出了ZR-0，一个拥有26亿参数的端到端VLA模型，它使用密集的具身思维链（ECoT）监督来对齐视觉-语言模型（VLM）中的交叉具身表示。ZR-0采用双流架构：预训练的VLM（系统2）在训练期间生成结构化的ECoT推理，而基于扩散Transformer的动作专家（系统1）通过流匹配产生连续的动作块。这两个组件通过交叉注意力耦合，并带有一个注意力掩码，该掩码限制动作专家仅输入提示特征，从而在推理时可以完全跳过ECoT生成而不损失任何性能。ZR-0在ProcCorpus-60M上进行了预训练，这是一个大规模数据集，包含来自超过40万条轨迹的大约6000万帧（约1000小时），密集的ECoT注释覆盖了96.8%的帧。我们在三个模拟基准上评估了ZR-0，涵盖了单臂（LIBERO）、双臂（RoboTwin 2.0）和类人（RoboCasa GR-1 Tabletop）具身形态，以及在xArm平台上的真实世界实验，展示了在所有设置中的强大性能。代码和模型检查点可在 https://github.com/RUCKBReasoning/ZR-0 获取。

## Abstract
Cross-embodiment transfer in vision-language-action (VLA) models remains challenging because low-level state and action spaces differ fundamentally across robot platforms. We observe that the high-level cognitive process underlying manipulation, including scene perception, object identification, task planning, and sub-task decomposition, is largely shared across embodiments. Based on this observation, we present ZR-0, a 2.6 billion parameter end-to-end VLA model that uses dense Embodied Chain-of-Thought (ECoT) supervision to align cross-embodiment representations within the vision-language model (VLM). ZR-0 adopts a dual-stream architecture: a pre-trained VLM (System 2) generates structured ECoT reasoning during training, while a Diffusion Transformer-based action expert (System 1) produces continuous action chunks via flow matching. The two components are coupled through cross-attention, with an attention mask that restricts the action expert to input prompt features only, enabling ECoT generation to be entirely skipped at inference without any performance loss. ZR-0 is pre-trained on ProcCorpus-60M, a large-scale dataset comprising approximately 60 million frames (approximately 1,000 hours) from over 400K trajectories, with dense ECoT annotations covering 96.8% of all frames. We evaluate ZR-0 on three simulation benchmarks spanning single-arm (LIBERO), bimanual (RoboTwin 2.0), and humanoid (RoboCasa GR-1 Tabletop) embodiments, as well as real-world experiments on the xArm platform, demonstrating strong performance across all settings. Code and model checkpoints are available at https://github.com/RUCKBReasoning/ZR-0.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：视觉-语言-动作（VLA）模型在不同机器人本体（embodiment）间的迁移（cross-embodiment transfer）面临根本性困难——不同机器人的低层状态/动作空间（自由度、控制接口、传感器配置等）差异巨大，导致模型容易过拟合到具体本体的模式，而难以学习共享的、可迁移的表征。
- **关键观察**：虽然低层物理空间因本体而异，但高层认知过程（场景感知、物体识别、任务规划、子任务分解）在不同本体间是高度共享的。因此，若能让模型在训练中通过对齐这些共享的高层推理来学习表征，则有望实现有效的跨本体迁移。
- **整体意义**：本文提出利用**密集的具身思维链（Embodied Chain-of-Thought, ECoT）监督**作为桥梁，使预训练的视觉-语言模型（VLM）学习到与本体无关的推理表征，从而显著提升VLA模型在多本体上的泛化能力，为构建通用的机器人策略模型提供新范式。

## 2. 提出的方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：采用双流架构（System 1 / System 2 认知框架）：
  - **System 2**（预训练VLM，初始化自 Qwen3-VL-2B-Instruct）：处理任务指令和图像观测，生成结构化的ECoT推理序列，包含场景描述、进度评估、未来计划、原子子任务动作、目标物体边界框和离散动作令牌。
  - **System 1**（基于Diffusion Transformer的动作专家）：通过流匹配（flow matching）生成连续动作块（action chunk），其输入为VLM的输出特征和机器人状态向量。
  - **连接方式**：VLM的最后一层隐藏特征通过交叉注意力传递给动作专家，且**注意力掩码限制动作专家仅能关注输入提示（指令+图像）的特征，而不关注ECoT令牌**。这使得推理时可以直接跳过ECoT自回归生成，仅需一次VLM前向传播即可获得动作专家所需特征，不损失性能。

- **关键技术细节**：
  - **ECoT组成**（6个组件，每个每帧标注）：
    - 场景描述（提升物体识别）
    - 进度评估（感知任务进度）
    - 未来计划（长时程推理）
    - 待执行动作（原子化子任务，如“抓取蓝色盘子”，与本体内存无关）
    - 目标物体边界框（视觉定位）
    - 离散动作令牌（连接高层推理与低层控制）
  - **训练损失**：联合优化两个目标：
    - 下一令牌预测损失（用于ECoT生成）：
      \( L_{\text{ntp}} = -\mathbb{E}_{\mathcal{D}} \left[ \sum_i \log \pi_{\theta'}(r_t^i \mid l, o_t, r_t^{<i}) \right] \)
    - 流匹配损失（用于连续动作生成）：
      \( L_{\text{fm}} = \mathbb{E}_{\mathcal{D},\tau,\epsilon} \left[ \| \pi_\theta(l, o_t, s_t, A_t^\tau, \tau) - (A_t - \epsilon) \|^2 \right] \)
    - 总损失：\( L = L_{\text{ntp}} + \alpha L_{\text{fm}} \)，其中 \(\alpha=5\)（预训练）或 \(1\)（后训练）。
  - **推理流程**：从高斯噪声初始化动作块，通过欧拉前向积分迭代去噪 \(N\) 步得到预测动作，不生成任何ECoT文本。在单张A6000 GPU上，生成一个动作块约需90ms，适合实时控制。

- **模型参数**：总计约2.6B（VLM 2.1B，动作专家0.5B）。动作块长度 \(H=32\)（预训练）或 \(10/16\)（后训练）。状态/动作维度统一填充至64维，并做min-max归一化。

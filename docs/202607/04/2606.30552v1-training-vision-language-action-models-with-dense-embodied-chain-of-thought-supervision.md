## 摘要
视觉-语言-动作（VLA）模型中的跨具身迁移仍然具有挑战性，因为不同机器人平台的低层状态和动作空间存在根本差异。我们观察到，操作过程中的高层认知过程，包括场景感知、物体识别、任务规划和子任务分解，在很大程度上是跨具身共享的。基于这一观察，我们提出了ZR-0，一个拥有26亿参数的端到端VLA模型，该模型使用密集的具身思维链（ECoT）监督来对齐视觉-语言模型（VLM）中的跨具身表示。ZR-0采用双流架构：一个预训练的VLM（系统2）在训练期间生成结构化的ECoT推理，而基于扩散Transformer的动作专家（系统1）通过流匹配生成连续的动作块。这两个组件通过交叉注意力耦合，并使用一个注意力掩码将动作专家限制为仅输入提示特征，从而可以在推理时完全跳过ECoT生成而不损失任何性能。ZR-0在ProcCorpus-60M上进行预训练，这是一个大规模数据集，包含来自超过40万条轨迹的大约6000万帧（约1000小时），密集的ECoT注释覆盖了所有帧的96.8%。我们在三个模拟基准上评估了ZR-0，涵盖单臂（LIBERO）、双臂（RoboTwin 2.0）和人形（RoboCasa GR-1 Tabletop）具身，以及在xArm平台上的真实世界实验，展示了在所有设置中的强劲性能。代码和模型检查点可在 https://github.com/RUCKBReasoning/ZR-0 获取。

## Abstract
Cross-embodiment transfer in vision-language-action (VLA) models remains challenging because low-level state and action spaces differ fundamentally across robot platforms. We observe that the high-level cognitive process underlying manipulation, including scene perception, object identification, task planning, and sub-task decomposition, is largely shared across embodiments. Based on this observation, we present ZR-0, a 2.6 billion parameter end-to-end VLA model that uses dense Embodied Chain-of-Thought (ECoT) supervision to align cross-embodiment representations within the vision-language model (VLM). ZR-0 adopts a dual-stream architecture: a pre-trained VLM (System 2) generates structured ECoT reasoning during training, while a Diffusion Transformer-based action expert (System 1) produces continuous action chunks via flow matching. The two components are coupled through cross-attention, with an attention mask that restricts the action expert to input prompt features only, enabling ECoT generation to be entirely skipped at inference without any performance loss. ZR-0 is pre-trained on ProcCorpus-60M, a large-scale dataset comprising approximately 60 million frames (approximately 1,000 hours) from over 400K trajectories, with dense ECoT annotations covering 96.8% of all frames. We evaluate ZR-0 on three simulation benchmarks spanning single-arm (LIBERO), bimanual (RoboTwin 2.0), and humanoid (RoboCasa GR-1 Tabletop) embodiments, as well as real-world experiments on the xArm platform, demonstrating strong performance across all settings. Code and model checkpoints are available at https://github.com/RUCKBReasoning/ZR-0.

### 论文详细总结

#### 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**： 视觉-语言-动作（VLA）模型在实现跨本体（cross-embodiment）迁移时面临根本性挑战。不同机器人平台（如单臂、双臂、人形）的底层状态（state）和动作（action）空间在维度、物理意义和控制方式上存在巨大差异，导致模型难以学习通用的、可迁移的策略。
- **研究背景**： 现有方法主要通过格式统一（如零填充、归一化）来缓解此问题，但未能解决**语义对齐**的深层挑战。即，模型学习到的仍是本体特定的模式，而非共享的、可迁移的表示。
- **核心洞察**： 作者观察到，尽管底层控制存在差异，但机器人执行操作所需的高层次认知过程（如场景感知、物体识别、任务规划、子目标分解）在不同本体间是高度共享的。这些本体无关的推理过程才是跨本体迁移应该捕捉的关键知识。

#### 2. 方法论：核心思想、关键技术细节

- **核心思想**： 提出ZR-0模型，通过引入**密集的具身思维链（Embodied Chain-of-Thought, ECoT）监督**，强制模型的视觉-语言部分学习到对齐的、可迁移的跨本体表示，而将本体特定的低层控制任务留给专门的专家模块。
- **关键技术细节**：
    - **双流架构（系统1 / 系统2）**：
        - **系统2（System 2）**： 预训练的视觉-语言模型（VLM），负责处理任务指令和视觉观测。在训练时，它需要生成结构化的ECoT推理序列。
        - **系统1（System 1）**： 基于扩散Transformer的动作专家，负责生成连续的、本体特定的动作块。
    - **耦合机制**： 两个系统通过**交叉注意力**连接。动作专家的DiT块以VLM的最后层隐藏状态作为key/value进行交叉注意力计算，从而将高层推理信息注入低层动作生成。
    - **ECoT的组成与作用**： 每个ECoT序列包含六个组件，覆盖了从感知到执行的全过程，其设计目标是强制VLM学习特定能力：
        - *场景描述*（Scene Description）：提升物体识别能力。
        - *进度评估*（Progress Assessment）：感知任务进度。
        - *未来计划*（Future Plan）：进行时空推理与长程规划。
        - *待执行动作*（To-Do Actions）：分解为不可知本体的原子子任务，是实现跨本体对齐的关键。
        - *目标物体*（Target Objects）：以边界框形式提供视觉接地监督，提升空间注意力。
        - *离散动作*（Discrete Actions）：由FAST分词器生成的本体特定离散动作，提供高层次推理与低层控制之间的桥梁。
    - **推理时零开销**： 通过一个精心设计的**交叉注意力掩码**，该掩码在训练时就限制动作专家只能关注VLM中对应于**输入提示（指令+图像）** 的特征。因此，在推理时，只需对输入进行一次前向传播获取特征即可，完全无需生成ECoT文本，避免了昂贵的自回归解码过程。
    - **训练目标**： 联合优化两个目标：
        - **下一个词预测损失**： 用于训练VLM生成ECoT。
        - **流匹配损失**： 通过去噪向量场预测，训练整个模型生成连续动作。
    - **数据集ProcCorpus-60M**： 大规模机器人数据集，包含约6000万帧（1000小时），来自40万条轨迹。关键特征是**96.8%的帧都带有自动生成的密集ECoT注释**，为模型提供了丰富的对齐信号。

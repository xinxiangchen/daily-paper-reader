## 摘要
视觉-语言-动作（VLA）模型已成为具身人工智能的重要范式。然而，现有VLA模型通常假设光照良好且稳定的室内环境，而实际具身操作可能涉及由光照变化引起的退化RGB观测，这给稳健的机器人操作带来了关键挑战。为解决这一问题，我们提出Event-VLA，一种事件增强的VLA框架，用于在变化光照条件下实现可泛化的操作。我们将可见性退化下的基于VLA的操作视为一个针对以RGB为中心的策略的实际鲁棒性问题，并引入事件流作为一种对光照鲁棒且对运动敏感的互补观测，以提升在不同可见度水平下的鲁棒性。具体来说，与直接将事件特征融合到全局语义标记空间中的传统多模态融合不同，Event-VLA通过一个动作查询路由路径注入事件信息。它使用可学习的动作查询从VLA推理过程中提取任务相关语义，并通过门控交叉注意力选择性聚合事件标记，以构建事件感知的动作表示。这种设计保留了预训练的RGB-语言语义先验，同时有效利用事件信息进行稳健的动作预测。在仿真和实际部署中的实验表明，Event-VLA在正常光照下保持强大的操作性能，并在低光照退化和接近黑暗的真实环境中提高了成功率。

## Abstract
Vision-Language-Action (VLA) models have become an important paradigm of embodied AI. However, existing VLA models typically assume well-lit and stable indoor settings, while real-world embodied manipulation may involve degraded RGB observations caused by illumination shifts, posing critical challenges for robust robotic manipulation. To address this gap, we propose \textbf{Event-VLA}, an event-enhanced VLA framework for generalizable manipulation across varying illumination conditions. We formulate VLA-based manipulation under degraded visibility as a practical robustness problem for RGB-centric policies, and introduce event streams as an illumination-robust, motion-sensitive complementary observation to improve robustness across visibility levels. Specifically, unlike conventional multimodal fusion that directly merges event features into the global semantic token space, Event-VLA injects event information through an action-query routing pathway. It uses learnable action queries to extract task-relevant semantics from the VLA reasoning process, and selectively aggregates event tokens via gated cross-attention to construct event-aware action representations. This design preserves the pretrained RGB-language semantic priors while effectively leveraging event information for robust action prediction. Experiments in simulation and real-world deployment show that Event-VLA maintains strong manipulation performance under normal lighting and improves success rates under low-light degradation and near-dark real-world settings.

- **背景**：事件相机（Event Camera）具有高动态范围、低延迟、对运动和光照变化敏感等特性，能在低光照和快速运动条件下捕捉稳定的动态视觉线索，因此可作为RGB的互补观测来增强VLA模型。
- **核心问题**：如何将异步高频的事件流有效地融入到预训练的VLA模型中，既能利用事件信息提升鲁棒性，又不破坏预训练的RGB-语言语义先验。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- Event-VLA 将事件信息作为“动作时间物理残差”来处理，而不是全局语义令牌。事件特征通过**动作查询路由（Action-Query Routing）** 路径注入，在预训练VLA生成语义上下文之后，通过**门控交叉注意力**选择性聚合事件令牌，构建事件感知的动作表示，实现了对预训练VLA语义先验的保留和事件信息的有效利用。

### 关键技术细节

#### a. PREI（Physical Residual Event Integration）事件表示
- 将异步事件流转换为紧凑的物理残差图，包含三个通道：
  - **Instantaneous（瞬时）**：强调最近局部变化。
  - **Salient（显著）**：突出局部显著事件活动。
  - **Persistent（持久）**：保留窗口内累积的事件结构。
- 公式：计算带衰减的activity map，归一化后得到三通道残差图 \(E_{prei}^t \in [0,1]^{H\times W \times 3}\)。

#### b. 事件编码器与特征蒸馏
- 将PREI图用编码器映射为VLA兼容的事件令牌 \(Y_e^t = [g_e^t; P_e^t]\)。
- 使用冻结的视觉教师（DINO分支）进行特征级蒸馏，对齐事件特征与RGB特征空间，但保持事件令牌在预训练VLA的全局语义上下文之外。

#### c. 动作条件事件接口（Action-Conditioned Event Interface）
- **输入到VLM backbone**：RGB令牌、文本令牌、本体感知令牌、动作占位符、可学习查询令牌（公共查询 \(Q_c^0\)、动作查询 \(Q_a^0\)、事件查询 \(Q_e^0\)）。
- **VLM backbone** 输出上下文化隐藏状态 \(H_t = [Z_{lm}^t; Q_c^t; Q_a^t; Q_e^t]\)。
- **门控交叉注意力融合**：事件令牌 \(Y_e^t\) 不进入VLM，而是与VLM隐藏状态进行交叉注意力，得到 \(Z_{attn}^t\)，再通过门控融合得到 \(Z_{fused}^t = \sigma \cdot Z_{attn}^t + Z_{lm}^t\)。
- **查询引导路由**：公共查询和动作查询将融合令牌路由到动作头；公共查询和事件查询路由到辅助事件头。
- **动作头** 预测未来动作块 \(\hat{A}_t\)。
- **辅助事件头** 预测未来PREI目标 \(\hat{E}_{prei}^{t+1:t+K}\)，仅作为训练正则化。

#### d. 训练目标
- 动作损失：\(\mathcal{L}_{act} = \frac{1}{K}\sum \|\hat{a}_{t+k} - a_{t+k}\|_1\)
- 未来事件重建损失 \(\mathcal{L}_{evt}\) 和导数一致性损失 \(\mathcal{L}_{deriv}\)，只在训练时使用。
- 总损失：\(\mathcal{L} = \mathcal{L}_{act} + \lambda_{evt}\mathcal{L}_{evt} + \lambda_{deriv}\mathcal{L}_{deriv}\)

1. **事件增强的有效性**：Event-VLA 在低光（LL-Severe）下平均成功率95.6%，而RGB-only基线（如π0）仅61.9%，证明事件流能显著提升退化光照下的鲁棒性。

3. **表示与接口的关键性**：PREI 表示优于时间表面；动作条件查询路由优于直接统一编码或适配器融合，且延迟更低（+2.157 ms vs +62.874 ms）。
4. **辅助正则化有效**：未来事件预测损失和导数一致性损失带来额外增益。
5. **真实世界验证**：在真实Franka机器人上，Event-VLA在近暗条件下达到52.5%成功率，而π0仅15%，OpenVLA-OFT仅12.5%。

## 7. 优点

- **设计理念新颖**：首次将事件流作为动作时间物理残差而非全局语义令牌注入VLA，实现了结构化的动作条件融合，保留了预训练先验。
- **轻量高效**：PREI 表示紧凑，门控交叉注意力和查询路由引入的延迟很小（~2ms），适合实时控制。
- **系统消融**：对表示、接口、查询、正则化进行了全面消融，提供了丰富的见解。
- **实验覆盖仿真和真实**：不仅有大规模的仿真benchmark，还有真实机器人部署验证，增加了说服力。
- **处理数据缺失问题**：LIBERO无原生事件流，作者设计了RGB-to-event模拟器并进行了严格泄漏控制，保证了实验公平性。

## 8. 不足与局限

- **模拟器依赖**：LIBERO-Cross 使用RGB-to-event模拟器生成事件，不能完全模拟真实事件传感器的噪声、触发抖动和校准误差，可能导致仿真到真实的差距。
- **真实实验规模小**：仅4个任务，且只在Franka Research 3机器人上进行，物体和环境多样性有限，泛化性有待验证。
- **硬件需求**：需要额外的事件相机及同步设备，增加了部署复杂度和成本。
- **光照退化类型窄**：主要针对低光照退化，未考虑运动模糊、遮挡、传感器故障等其他退化类型。
- **未探索与本体感知或触觉的融合**：Event-VLA仅融合了事件与RGB/语言/本体，未与其他模态（如触觉）结合。
- **长期任务未验证**：实验集中在单步骤或短时程操作（动作块大小8），长程操作中的稳健性未知。

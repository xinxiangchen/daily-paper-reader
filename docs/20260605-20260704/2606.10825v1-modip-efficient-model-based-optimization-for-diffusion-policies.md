## 摘要
扩散策略（DPs）已成为机器人学习中的表达性策略表示，通常与行为克隆（BC）等模仿学习方法一起使用。然而，尽管它们的成功主要局限于BC，直接进行强化学习（RL）微调仍然具有挑战性，因为动作是通过多步去噪过程生成的。在这项工作中，我们提出了MODIP，一个用于扩散策略的离线到在线微调框架。MODIP不是直接将RL应用于DPs，而是利用世界模型（WM）来指导策略适应，并保持BC的简单性和稳定性。我们利用模型预测控制（MPC）在WM内生成高质量轨迹，并将其作为监督目标来微调DP。为了使MPC规划高效，MODIP使用终端状态值而非依赖于策略的状态-动作值，从而减少推理时间。此外，MODIP使用与策略无关的TD目标训练评论家，从而减少训练时间。在D4RL（MuJoCo、Kitchen）和RoboMimic任务上的实验表明，MODIP改进了超出BC的扩散策略，并且与扩散策略RL微调方法和强基于模型基线（如TD-MPC2）相比具有竞争力或更优表现。

## Abstract
Diffusion policies (DPs) have emerged as expressive policy representations for robot learning, often used with imitation learning methods such as behavioral cloning (BC). However, while their success has largely been confined to BC, direct reinforcement learning (RL) fine-tuning remains challenging because actions are generated through a multi-step denoising process. In this work, we propose MODIP, a framework for the offline-to-online fine-tuning of DPs. Rather than directly applying RL to the DPs, MODIP leverages a world model (WM) to guide policy adaptation and keeps the simplicity and stability of BC. We utilize model predictive control (MPC) to generate high-quality trajectories within the WM, and use them as supervised targets for fine-tuning the DP. To make MPC planning efficient, MODIP uses a terminal state value instead of a policy-dependent state-action value, reducing inference time. Additionally, MODIP trains critics with policy-independent TD targets, reducing training time. Experiments on D4RL (MuJoCo, Kitchen) and RoboMimic tasks show that MODIP improves diffusion policies beyond BC, and is competitive with or outperforms diffusion policy RL fine-tuning methods and strong model-based baselines such as TD-MPC2.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
不使用RL目标直接优化DP，而是通过MPC在世界模型中规划改进轨迹，再用BC损失（去噪损失）将轨迹蒸馏回DP，保持监督学习的简单性。

### 关键技术细节
1. **MPC-to-Policy蒸馏**：
   - 离线阶段：先用BC预训练DP。
   - 在线微调阶段：DP作为MPC规划的提议先验，MPC在潜空间（由编码器、潜动态模型、奖励模型组成）中优化动作序列，生成改进轨迹存入在线缓存。
   - 策略更新：使用去噪BC损失 `L_DP(ϕ) = E[||ϵ - ϵ_ϕ(a_k, k, h_η(s_t))||^2]` 训练DP，目标动作序列来自MPC规划轨迹（而非专家数据）。同时用冻结的BC策略定期望化，稳定过渡。

2. **高效混合规划**：
   - 使用MPPI（Model Predictive Path Integral）进行轨迹优化。
   - 提议分布混合：部分来自DP生成的高似然动作序列，部分来自高斯分布随机采样。
   - 终端值函数：使用状态值函数V(z)而非常见Q(z, π(z))，避免在终端状态调用多步去噪过程，显著降低推理时间。规划目标：
     ```
     A* = argmax [Σ γ^k R(z_{t+k}, a_{t+k}) + γ^{H_mpc} V(z_{t+H_mpc})]
     ```
     其中z为潜状态，H_mpc为规划视界。

3. **策略无关的评论家学习**：
   - 将评论家学习（状态值V、优势A）与当前DP解耦：TD目标使用目标状态值V(s')，而非采样DP动作。目标：
     ```
     y_t = r_t + γ(1-d_t) min_j V_ψj(h_η(s_{t+1}))
     ```
   - 训练V、Q和优势网络，使用EMA目标网络。避免在每次评论家更新时运行DP去噪过程，降低训练成本。

### 算法流程
- 预训练：在离线数据上训练扩散策略、世界模型（编码器、潜动态、奖励模型）和评论家（V、Q、优势）。
- 在线微调：每步使用DP引导的MPPI规划动作，执行后存储轨迹到在线缓存；从混合缓存（离线+在线）采样，用BC损失更新DP；世界模型和评论家也持续更新；使用时变混合系数和正则化权重。

## 6. 论文的主要结论与发现

- MODIP在D4RL MuJoCo上超越所有基线（包括PA-RL、TD-MPC2）；在Kitchen上达到最高成功率（Complete 0.94, Partial 0.98）；在RoboMimic上与最强方法DSRL相当或略有差距。

- 使用终端状态值V代替Q(s,π(s))不仅降低推理时间2.9倍，还提升了最终性能。
- 策略无关评论家训练节省约1.6倍总训练时间（40h→25h），且不影响性能。
- 扩散策略作为MPC先验优于高斯MLP策略，表明多模态表达性的优势。
